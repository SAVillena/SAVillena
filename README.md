### 🚀 Sobre mí

Perfil orientado a resolver problemas reales a través de la tecnología, dominando el ciclo completo de desarrollo. Me especializo en:

*   **Desarrollo Full-Stack:** Creación de aplicaciones web modernas y de alto rendimiento (Astro, Next.js, React).
*   **Inteligencia Artificial y Automatización:** Diseño de flujos logísticos 24/7 usando `n8n`, LLMs (OpenAI, Gemini) y agentes de atención (Chatwoot, Evolution API).
*   **Sistemas Geoespaciales (GIS):** Manejo de datos territoriales, telemetría y mapas usando `PostGIS` y bases de datos relacionales avanzadas.
*   **Infraestructura:** Despliegue, gestión y monitoreo en entornos Linux, Docker y clusters Proxmox VE.

### 💼 Experiencia Destacada

- 🤖 **Ingeniero Freelance:** Diseño e implementación de ecosistemas digitales, automatización logística con n8n e IA para atención 24/7 (Grúas AMS), y sistemas premium de reserva.
- 🌍 **INNTECH (Ingeniero de Software y TI):** Desarrollo de *DustSense*, plataforma privada de monitoreo ambiental geoespacial para minería, midiendo concentraciones PM2.5/PM10 en tiempo real con alertas tempranas usando el stack PERN + PostGIS.
- 🏢 **Mundo Pacífico (Desarrollador Core):** Automatización del inventario de infraestructura, conectando con la API de Proxmox para auditar +400 VMs, clusters y almacenamiento físico a escala empresarial.
